# Fuck_QCZJ

青春浙江-青年大学习打卡，仅浙江地区

## 使用方法
1、打开main.py

2、修改nid、cardNo、openid，其中nid和openid需要自己抓包

3、运行即可

nid是团组织的代号
cardNo是打卡用的昵称
openid是微信分配给你的识别码
